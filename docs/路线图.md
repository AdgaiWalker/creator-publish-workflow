# 路线图

这份路线图只记录后续候选方向，不代表已经实现，也不代表已经进入当前主流程。

## 候选方向

### 创作者反馈整理器

状态：

- 候选
- 暂不实现
- 未完成验证

定位：

- 把作品发布后的评论、回复、issue、私信和讨论区内容，整理成可行动的信息
- 将外部反馈回流到 README、共用底稿和下一轮分发

为什么暂时不做：

- 当前没有足够时间进入实现与验证
- 反馈来源、整理模式和回流规则还没有经过真实使用验证
- 现在对外主线已经足够清楚，继续把未验证能力放进主流程会显得超前

后续再做时，优先检查：

- 反馈来源是否足够稳定
- 输出是否真的能回流到 README、底稿和下一轮内容
- 适配范围是否只做 X，还是一开始就覆盖多平台
